🎮 게임 소개

1989년 프랑스의 델핀 소프트웨어(Delphine Software)에서 개발한 《미래 전쟁(Future Wars : Adventures in Time)》은 독창적인 그래픽과 시네마틱한 연출로 고전 게임 팬들의 기억에 남아있는 명작 어드벤처 게임입니다. 기존의 텍스트 타이핑 방식에서 탈피하여, 마우스 클릭과 팝업 명령어를 활용한 혁신적인 인터페이스를 선보였습니다.

SCENE 1  현재 (1989) — 청소부의 발견
SCENE 2  중세 (1304년) — 성 탈출 작전
SCENE 3  미래 (44세기) — 외계 함선 침투

📜 줄거리 요약 (타임 트래블)

평범하게 고층 빌딩의 유리창 청소부로 일하던 주인공(Hero)은 일 도중 우연히 건물 내부에서 비밀스러운 타임머신을 발견하게 됩니다. 이 우연한 발견을 시작으로 주인공은 과거와 미래, 시공간을 넘나들며 지구를 파멸시키려는 외계 세력의 위협에 맞서 인류를 구하는 거대한 여정에 휘말리게 됩니다.

🛠️ 한국어판 특징 및 플레이 팁

  • 완벽한 몰입감: 한글화하는 사람들의 모임(한사모)의 장인정신이 담긴 번역으로 난해한 SF 설정과 스토리를 완벽하게 한국어로 즐길 수 있습니다.
  • 높은 난이도: 당시 고전 게임 특유의 불친절함과 픽셀 단위의 철저한 아이템 탐색이 요구되므로, 세밀한 마우스 컨트롤과 관찰력이 필수입니다.
  • 상징적인 요소: 과거 복제 방지용으로 악명 높았던 '암호표' 시스템과 영화 같은 오프닝 연출은 이 게임의 가장 큰 매력 포인트 중 하나입니다.

게임 내 메시지 박스 배경색 · 폰트 색상을 korean.ini에서 0~15 인덱스로 자유롭게 설정합니다.

8 연노랑
9 주황

11 매우어두운진한빨강
12 연두
13 밝은노랑

15 하늘색
# 액션/시스템/확인 메뉴 (SelectionMenu 공통) menu_bg_color = 14 # 14=회색(기본) / 3=밝은빨강 / 11=매우어두운진한빨강 추천 menu_text_color_normal = 2 # 비선택 항목 글씨 menu_text_color_selected = 1 # 선택(하이라이트) 항목 글씨 # 행동 실패 메시지 failure_bg_color = 7 # 4=마젠타(기본) / 7=남색 failure_text_color = 2 # 시스템 알림 메시지 other_bg_color = 14 other_text_color = 2 # 하단 커맨드라인 메시지 창 cmd_bg_color = 0 # 0=검정(기본) / 1=흰색 배경 cmd_border_color = 2 cmd_text_color = 2
🔴 failure_bg_color = 4 (마젠타)
  • 행동 실패 시 마젠타(원본 스타일) 배경
  • 가장 원본에 가까운 설정
  • 흰색 글씨와 높은 대비
🔵 failure_bg_color = 7 (남색)
  • 진한 남색 배경으로 차분한 느낌
  • 눈에 덜 피로한 대안 설정
  • 메뉴와 통일감 있는 색상
⬛ cmd_bg_color = 0 (검정)
  • 하단 바 검정 배경 (기본값)
  • 게임 화면과 자연스러운 연결
  • 원본 DOS 분위기 유지
⬜ cmd_bg_color = 1 (회색빛)
  • 하단 바 밝은 배경
  • 텍스트 가독성 향상
  • 현대적인 UI 느낌
